# Trainings-app
Trainings app is a web application built using ReactJS and ExpressJS frameworks. It allows users
to browse and join courses. The app is hosted on Vercel and can be accessed at: https://ing-app.vercel.app/

# Technologies
* ReactJS
* ExpressJS
* MySQL
* Jest
